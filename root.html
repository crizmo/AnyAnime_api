Hi! Welcome to AnyAnime API

AnyAnime is a free API that allows you to get random anime images.


How to use it?

It's very simple!

Just use the following endpoint:

  • /v1/anime/:type/:number (number is optional)
  • /v1/anime/:type/:number (number is optional)

Where :type is the type of image you want to get. It can be gif or png.

And :number is the number of images you want to get. It can be any number between 1 and 10.


Note: If the number is not provided, it will return 1 image by default.
Max number is 10.


Examples

  • /v1/anime/gif/2
  • /v1/anime/png/3

Note: If you are an old user, please update your code to use the new endpoint.
